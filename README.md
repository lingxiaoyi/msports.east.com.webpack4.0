
# webpack4.0多页面项目

- 此套构建项目由webpack3.0升级到4.0的配置修改
- 技术栈：webpack4.0 ES5 ejs生成模板
- 已实现登录注册接口，文章增删改查接口
- 用eslint做代码规范,将代码统一化

### 一、学习使用


git clone

```
git clone https://github.com/lingxiaoyi/msports.east.com.webpack4.0.git
```

1.1.安装

```
cnpm install
npm run start 本地运行
npm run buildtest 查看本地打包代码
```
